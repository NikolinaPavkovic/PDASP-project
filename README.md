# PDASP-project
## STEPS
### Start the network

Go to folder <b>fabric-samples/car-application</b>

then run following script:
```bash
./startFabric.sh
```

### Start application
Next step is to start the application which calls chaincode

Go to folder <b>fabric-samples/car-application/go</b>

then run following script:
```bash
./runfabcar.sh
```
